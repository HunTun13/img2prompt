Support and feedback

Contact Img2Prompt

Found a bug, have a feature request, or need to ask about privacy? The project issue tracker is the fastest public support channel.

Open a support issue

Use GitHub Issues to report a problem or suggest an improvement. Please check whether a similar issue already exists before opening a new one.

Img2Prompt issue tracker

Public reports help us keep fixes organized and let other users see known problems.

Go to GitHub Issues

What to include

  • what you expected to happen and what happened instead;
  • the browser and device you used;
  • the selected prompt format and detail level;
  • the exact error message, if one appeared.

Protect your information

GitHub Issues are public. Never include API keys, passwords, confidential images, private image URLs, or personal information in a report. A cropped screenshot of the error is usually enough.

Privacy questions

Label your issue as a privacy question and do not attach the image concerned. Read the Privacy Policy first for details about image and service-data processing.
